# Laboration 1, DT208G Grunderna i TypeScript
Detta är en laboration i _Webbutveckingsprogrammet_ på Mittuniveristetet.

## Länk till webbplats
[Min webbplats](https://github.com/Safe-Pizza/DT208G_Laboration1)

## Syfte
Syftet med uppgiften är att förstå vad Typescript är, och hur det skiljer sig från JavaScript.
Förstå hur TypeScript transpileras till JavaScript och kunna skriva grundläggande TypeScript-syntax för TypeScript för webbläsare samt kunna använda byggverktyget Vite för att skapa och köra ett TypeScript-projekt.

 ## Uppgift
Uppgiften för denna laboration är ett skapa en enkel webbapplikation för att hantera högskolekurser.
Applikatinen ska kodas med TypeScript, localStorage ska användas för att lagra kurser mellan sessioner och DOM-manipulation ska användas för visa kurserna på sidan.
